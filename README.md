
# Skills

## langwiki-okf

从本地文件（PDF/Word/Excel/PPT/Markdown/代码等 16 种格式）或远程 API 生成 OKF（开放知识格式）知识包。

核心能力：
- 16 种文件格式支持（PDF/Word/Excel/PPT/Markdown/代码/配置/HTML/CSV）
- 多 LLM 后端：Gemini/Claude/OpenAI/DeepSeek/通义千问/Ollama
- vendor 内置源码，无需 git clone 上游仓库
- 支持 `cli`/`api`/`auto` 三种编排模式

详见 [langwiki-okf/SKILL.md](langwiki-okf/SKILL.md)。

## 安装

```bash
# 将 skill 安装到 CodeBuddy
cp -r langwiki-okf ~/.codebuddy/skills/

# 或从 zip 安装
# 在 CodeBuddy 中执行: install skill from langwiki-okf.zip
```
